# 🛒 FullStack E-Ticaret Projesi

Bu proje, **ASP.NET Core 9 Web API**, **React.js** ve **Microsoft SQL Server** kullanılarak geliştirilmiş **full-stack bir e-ticaret uygulamasıdır**.

Projede modern web uygulama mimarisi kullanılarak **frontend ve backend katmanları birbirinden ayrılmış**, kullanıcıların ürünleri görüntüleyebildiği, sepete ekleyebildiği ve güvenli şekilde ödeme yapabildiği bir alışveriş sistemi geliştirilmiştir.

Ayrıca uygulamada **Redux Toolkit ile state yönetimi**, **JWT ile kimlik doğrulama** ve **iyzico ödeme entegrasyonu** kullanılarak gerçek dünya e-ticaret sistemlerine yakın bir yapı oluşturulmuştur.

---

# 🏗 Proje Mimarisi

Uygulama aşağıdaki mimari yapıya sahiptir:

- Frontend: **React.js**          
- Backend: **ASP.NET Core 9 Web API**      
- Veritabanı: **Microsoft SQL Server**

### Ek kullanılan teknolojiler

- **Redux Toolkit** → Frontend state yönetimi  
- **JWT Authentication** → Kullanıcı güvenliği  
- **iyzico API** → Online ödeme altyapısı

 ---

# 🚀 Özellikler

## 🔐 Kimlik Doğrulama ve Yetkilendirme
- **JWT (JSON Web Token)** kullanılarak güvenli kullanıcı giriş ve yetkilendirme sistemi oluşturuldu.

## 🛍 Ürün Yönetimi
- Kullanıcılar ürünleri listeleyebilir.
- Ürün detaylarını görüntüleyebilir.

## 🛒 Sepet Sistemi
- Kullanıcılar ürünleri sepete ekleyebilir veya sepetten kaldırabilir.
- Sepet üzerinden sipariş oluşturabilir.

## 💳 Online Ödeme Altyapısı
- **iyzico ödeme sistemi entegrasyonu** ile güvenli online ödeme altyapısı oluşturuldu.
- Kullanıcılar kredi veya banka kartı ile ödeme yapabilir.

## ⚡ State Management
- **Redux Toolkit** kullanılarak uygulama genelinde state yönetimi sağlandı.
- Sepet verileri, kullanıcı bilgileri ve arayüz durumları merkezi olarak yönetildi.

## 🔄 RESTful API Mimarisi
- Backend tarafında **ASP.NET Core Web API** kullanılarak RESTful servisler geliştirildi.
- Frontend ve backend veri iletişimi API üzerinden sağlandı.

## 🧩 Veritabanı Tasarımı
- **Microsoft SQL Server** üzerinde **Entity Framework Core ile** ürün, kullanıcı, sipariş ve sepet verilerini yöneten ilişkisel veritabanı tasarlandı.
